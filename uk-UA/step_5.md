Редагуйте HTML та CSS для ** Ігрова кімната **, щоб веб-сторінка виглядала так:

![скріншот](images/rooms-games-challenge.png)

Hint: You'll need to add an `<a>` link in `gamesroom.html` which links to `index.html`.
